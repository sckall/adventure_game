# 🎮 Chui的冒险 - 2D平台冒险游戏

> 用Godot 4.6重新制作的完整2D平台冒险游戏！

## 📦 游戏特色

### 🎯 核心玩法
- **5个精心设计的关卡** - 草原、森林、山地、洞穴、城堡
- **击杀所有怪物** 才能到达终点门
- **收集物品** 获得蘑菇、绿瓶、黄瓶奖励
- **关卡解锁系统** - 通关后自动解锁下一关

### 🎮 战斗系统
- **多种敌人类型** - 史莱姆、蝙蝠、骷髅、刺猬、蜗牛、AI Boss
- **敌人AI** - 视野内自动追踪玩家
- **受击击退** - 怪物受伤会被击退
- **坠落死亡** - 怪物掉出地图边界会死亡

### 🏪 商店系统
- **升级系统** - 消耗蘑菇提升属性
- **可升级项目**：
  - ❤️ 生命值提升 (HP+1/级)
  - 🏃 移动速度提升 (+10速度/级)
  - 🦘 跳跃力提升 (+30跳跃/级)
  - ⚔️ 攻击力提升 (+0.5伤害/级)

### 💾 存档系统
- **3个存档槽** - 独立保存游戏进度
- **自动保存** - 通关后自动保存
- **数据包含** - 关卡、角色、收集品、升级等级

### 🎨 画面效果
- **动态背景** - 根据时间变化的背景色
- **粒子特效** - 收集、死亡、受伤特效
- **平滑动画** - 角色变形、挤压效果
- **流畅物理** - 优化的平台跳跃手感

## 🎮 操作说明

### 基础操作
| 按键 | 功能 |
|------|------|
| A / D | 左右移动 |
| 空格 | 跳跃（可二段跳） |
| E | 互动 |
| K | 攻击 |
| L | 特殊技能 |
| P | 暂停 |

### 技能快捷键（根据角色不同）
| 按键 | 技能 |
|------|------|

| L | 震地猛击（战士） |
| 空中L | 冲刺（刺客） |
| L | 浮空（法师） |
| L | 治疗（牧师） |
| L | 护盾（牧师） |
| L | 钩锁（射手） |

## 🎯 游戏目标

### 通关条件
1. **击杀所有怪物** - 只有击败所有敌人才会出现胜利条件
2. **到达终点门** - 击败敌人后到达金色传送门
3. **获得奖励** - 收集的物品会计入总奖励

### 收集品
| 物品 | 分数 |
|------|------|
| 🍄 蘑菇 | 50分/个 |
| 🟢 绿瓶 | 100分/个 |
| 🟡 黄瓶 | 200分/个 |

## 🎯 关卡设计
